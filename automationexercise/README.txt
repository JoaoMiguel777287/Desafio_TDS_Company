# Projeto de Testes Automatizados - Robot Framework

Automação de testes para o site [AutomationExercise](https://www.automationexercise.com) utilizando **Robot Framework** e **SeleniumLibrary**.

## 📁 Estrutura

robotframework_automationexercise/
├── tests/
│   ├── login.robot
│   └── cadastro.robot
├── pages/
│   ├── login.robot
│   └── cadastro.robot
├── resources/
│   └── common.robot
├── results/                
├── README.md               ✅ Explica tudo sobre o projeto.
├── requirements.txt        ✅ Contém as dependências para instalação.
├── Casos de testes.xlsx    ✅ Contém os cenários de testes manauis.
├── Plano de teste.txt      ✅ Descreve todo o escopo, a abordagem e outros detalhes do plano de testes.
